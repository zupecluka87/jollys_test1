# JOLLY-S ultra minimalistična stran

## Objavi na Netlify
1. Naloži mapo v GitHub repozitorij.
2. Na Netlify izberi "New site from Git".
3. Publish directory naj bo `/` oziroma root.
4. V Netlify vključi Identity.
5. Vključi Git Gateway.
6. V Identity povabi svoj e-mail.
7. Admin odpreš na `/admin`.

## Urejanje
Vse osnovne stvari urejaš v `/admin`:
- teksti
- projekti
- storitve in podstrani
- materiali / moodboard
- slike
- barve
- fonti
- kontakt
- logo monogram

## Velikosti slik
V adminu so pri vsakem polju za slike zapisane priporočene velikosti.
